# Changelog

## [Unreleased]

### Added
- Priority levels (high, medium, low) for tasks
- Tags support for task categorization
- Search functionality across title, description, and tags
- Filter functionality by status, priority, and due date
- Sort functionality by title, priority, and due date
- Due date support for tasks
- New CLI commands: search, filter, sort
- Performance validation for search and sort operations
- Integration tests for all new features

- Updated Task model to include priority, tags, and due date
- Enhanced TaskManager with search, filter, and sort methods
- Updated CLI interface to support new features
- Improved table display to show priority, tags, and due dates
- Updated unit and integration tests to cover new functionality
